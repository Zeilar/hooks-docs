# Installation

Apart from being super slim at less than **3kb gzipped** and having **0 dependencies**, this library ships with its own TypeScript declaration files.  
Simply install the package and you're good to go! 🚀

**npm:**

```bash
npm i @zeilar/hooks
```

**yarn:**

```bash
yarn add @zeilar/hooks
```

# ❗ Prerequisites ❗

-   NodeJS >= 10
-   React >= 17
